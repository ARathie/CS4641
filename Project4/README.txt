To run the experiments used in this analysis, you must download the CMU Reinforcement Learning Simulator at https://www.cs.cmu.edu/~awm/rlsim/

To run the simulator:
On windows: Start command prompt,
On Linux: Start shell.
Change directory to directory where files are extracted. Then Change directory to go in RL_sim directory.
Execute the command 'java -jar rl_sim.jar'
